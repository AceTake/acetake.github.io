Here are some resources to support you along your journey:  

 

W3 Schools is a great online resource to help you to build your confidence and skill in various programming languages that we, at HotBeansWe, utilise on a day to day basis, such as: HTML, CSS, JavaScript, Python and many more!       

Skilshare is a vital resource if you need to learn any specific topic as their dedicated courses allow you to explore specific things that interest yo, for example, making a website using Django and Python.

There are multiple great channels on YouTube that allow you to push your coding knowledge to new highs, one of our personal favourites at the office is TechWithTim who creates extremely high quality content for multiple oding languages such as Python.

Khan Academy is an excellent learning resource which provides easy to follow guides for many aspects of some of the most popular programming languages such as Python, HTML, CSS, JavaScript and more!

Udemy is a course selling platform which allows you to purchase courses from many experieced programmers, these courses are paid for individually, but they are also more advanced than similar courses on platforms such as skillshare.

CodeAcademy is a greatly useful tool to have in your programming toolkit, it provides detailed education on many of the programming language that we use at HotBeansWeb.
